# Focus Finder

Allows users to find a next or previous focusable element in a given direction relative to an element or an area.

### Installation

```smalltalk
Metacello new
   baseline: 'BlocPacFocusFinder';
   repository: 'github://feenkcom/bloc-pac:main/src';
   load
```
